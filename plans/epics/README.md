# Epic Plans

This folder holds broad plan folders that are still too large to hand directly to `feature-implementing`.

- Epics are refinement inputs for `program-refining`.
- `program-refining` turns an epic into bounded child feature seeds.
- Refined, implementation-ready child plans belong in `plans/{feature_name}/`.
- Finished plans belong in `plans/completed/{feature_name}/`.
